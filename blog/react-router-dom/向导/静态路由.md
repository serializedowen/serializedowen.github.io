### 静态路由

以前版本的 React Router 使用静态路由来配置应用程序的路由。这样可以在渲染之前检查和匹配路线。由于 v4 转移到动态组件而不是路由配置，因此一些以前的用例变得不那么明显和棘手。我们正在开发一个用于静态路由配置和 React Router 的软件包，以继续满足这些用例。现在正在开发中，但我们希望您能尝试一下并提供帮助。
